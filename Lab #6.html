# 3D
obs_sig = stats.norm.cdf(1)
print(stats.poisson.ppf(obs_sig,mu = 250))

266.0

Clearly seen that the single sigma uncertainty for the true signal strength is around 16.

3E. To see if it is symmetric, we can take the single sigma value vs then negative sigma value. If the difference between the sigma value and the mean of 250 is the same, then they are symmetric and not biased. If the values do not match, then it is biased.

In [56]:
obs_sig = stats.norm.cdf(1)
print(stats.poisson.ppf(obs_sig,mu = 250))
obs_sig_neg = stats.norm.cdf(-1)
print(stats.poisson.ppf(obs_sig_neg,mu = 250))

266.0
234.0

Both values are 16 away from the mean, meaning a symmetric uncertainty for true signal.

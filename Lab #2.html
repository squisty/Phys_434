# 4C

The signal must be around .77 values brighter for a trials factor of 10k, which isnt that much considering if a single 5 sigma event has a false occurence of 1/3.5 million, the 5 sigma event for a false occurence would be 1/(3.5 million *10000) or whatever the trials factor is. Applying this to the classic distribution gives a large signal value with a tiny chance to occur falsly.
